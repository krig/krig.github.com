most meta wins

The future will be grand, when we can all write C++0x and be meta-happy in realtime. No more guessing template types only to have the compiler go "No no, the correct type is this<convoluted,construction, of<template, arguments> >. Hooray for the less annoying compiler-animus.

templates

Some people mistakenly think that C++ is all about object orientation and inheritance and that whole bag. Not so, at least not anymore. Inheritance is old-fashioned, but not in the cool way like something that was out of style but is now back in. No, it's old-fashioned more like syphilis and slavery.

Don't write object oriented code in C++. Think generics. And RAII. That's where happiness lies. And if you think templates are too convoluted and too complicated, well, you're right. And it doesn't matter.
